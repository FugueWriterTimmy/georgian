Georgian (ქართული ენა) is a Kartvelian language spoken by almost 4 million people in Georgia, a small country in the Caucasus. Georgian has a unique, beautiful, flowing script and is renowned among linguists for rare features such as split ergativity, ejective consonants, and polypersonal agreement. This website is a set of tools and links that I either made or found useful that I hope will help you learn it. Please remember that this website is an active Work In Progress, and may therefore contain some mistakes, which I am actively working to remove.
